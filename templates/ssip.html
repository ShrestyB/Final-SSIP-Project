Steps

  • 1. QR Code Scan:
    Users scan the QR code to access the feedback form.
  • 2. Feedback Submission:
    Fill out the feedback form with your responses.
  • 3. Data Storage:
    Your feedback is securely stored in the MongoDB Cloud.
  • 4. Analysis & CSV:
    Data is analyzed using Python, creating separate CSV files for each district.
  • 5. Visualization:
    Python generates visual representations of district-wise feedback.
  • 6. Front-end Display:
    Interactive front-end displays district data and pie charts.
